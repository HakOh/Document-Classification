# Document Classification

### 개발 목표
결재문서를 스스로 분류해주는 시스템을 통해, 업무 효율과 시간 단축을 도모할수 있음.

### 개발 내용
- 정보공개포털(open.go.kr)의 결재문서들을 Web Crawler를 이용하여 수집
- CNN 모델을 통하여 모든 결재문서들을 분류

### 개발 환경
- aws - p2.xlarge
- ubuntu
